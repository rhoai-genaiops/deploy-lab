Graduate Learning Outcomes

Graduates will be able to:

1

Circuit & System Design

Design digital/analog circuits and embedded systems from requirements to prototype/production.

2

Architecture Analysis

Specify & analyze computer architectures and real-time systems using quantitative metrics.

3

Software Development

Develop reliable software in C/C++/Python/HDL with unit/integration testing and CI/CD.

4

Secure Systems

Deploy networked, secure systems; reason about risk, privacy, and resilience.

5

Signal Processing

Apply signals, control, and DSP to mechatronics/IoT/communications.

6

System Verification

Model & verify systems using simulation, formal methods, and hardware debug tools.

7

Professional Skills

Collaborate effectively, communicate design decisions, and manage engineering projects ethically.

8

Lifelong Learning

Learn independently and adapt to emerging technologies.

Laboratories & Infrastructure

Hardware Labs

FPGA boards (Xilinx/Intel), ARM Cortex-M dev kits, logic analyzers, oscilloscopes, SDRs, robot kits, sensor suites, power supplies.

Software Tools

HDL simulators (Verilator/ModelSim), EDA flows, MATLAB/Octave, Python (NumPy/SciPy), C/C++, GCC/Clang, CMake, Docker/Podman.

Development Environment

Kubernetes student cluster, GitLab/GitHub CI/CD, version control, code reviews, unit/integration testing.

Best Practices

Pair programming, documentation standards, static/dynamic analysis, test-driven development.

Capstone Design Examples

Autonomous Rover

Perception (embedded vision), path planning, motor control, V2X communications, safety case (ISO 26262-inspired), fleet operations demo.

Wearable Health Patch

Low-power sensors, BLE, embedded ML arrhythmia detection, privacy & security model, IEC 62304-inspired software lifecycle.

FPGA CNN Accelerator

RTL or HLS design, systolic array, quantized model, PCIe/AXI integration, throughput/latency/energy benchmarking.

SDR-Based Communication Link

Software-defined radio implementation with custom protocols, signal processing, and real-time performance optimization.
