AI501 - Advanced Generative AI Systems

Advanced Course, 15 ECTS

A comprehensive advanced course covering production-ready generative AI systems, from prompt engineering to enterprise deployment and security.

üìÑ Download Course Syllabus (PDF)

Course Overview

Course Code: AI501
Duration: 1 Semester (Advanced Level)
Total Credits: 15 ECTS
Prerequisites: Programming fundamentals, Basic ML/AI knowledge
Delivery: Lectures (L), Hands-on Labs (P), Discussions (D), Projects (S)
Assessment: Continuous evaluation, practical projects, final capstone

Core Focus Areas

  • Generative AI Fundamentals
  • Prompt Engineering & Optimization
  • Production AI Systems
  • Retrieval Augmented Generation
  • AI Security & Guardrails
  • Model Evaluation & Monitoring
  • Tool Integration & Agents
  • Multi-modal AI Systems
  • Model Optimization
  • Enterprise AI Deployment

Learning Outcomes

Upon completion, students will be able to:

1

GenAI Mastery

Design and implement sophisticated generative AI applications using state-of-the-art models and techniques.

2

Prompt Engineering

Master advanced prompt engineering techniques to optimize model behavior and output quality.

3

Production Systems

Build scalable, production-ready AI systems with proper versioning, testing, and deployment practices.

4

RAG Implementation

Implement retrieval-augmented generation systems for grounding AI responses in real knowledge bases.

5

AI Security

Apply security best practices, guardrails, and monitoring to ensure safe AI system deployment.

6

Multi-modal AI

Work with multi-modal models integrating text, image, audio, and other data modalities.

7

Model Optimization

Optimize models through quantization, compression, and fine-tuning for specific use cases.

Course Modules

Foundation Modules (1-3)

Module 1: AI Orientation

  • Basic understanding of use cases
  • Introduction to Generative AI
  • Course overview and roadmap

Module 2: AI Linguistics

  • Changing model behavior
  • Advanced prompt engineering
  • Prompt optimization techniques

Module 3: Ready to Scale 101

  • Introduction to Llama Stack
  • Introduction to GitOps
  • Prompt templates & versioning

Production Modules (4-7)

Module 4: Ready to Scale 201

  • Continuous model evaluation
  • Prompt/settings promotion
  • Test, promote, repeat cycles

Module 5: RAG Implementation

  • Grounding model responses
  • Document ingestion pipelines
  • Operationalizing RAG systems

Module 6: Guardrails

  • Safety mechanisms
  • Content filtering
  • Bias detection and mitigation

Module 7: Observability

  • Monitoring AI systems
  • Performance metrics
  • Alerting and diagnostics

Advanced Modules (8-11)

Module 8: Tool Calling & Agents

  • Tool calling mechanisms
  • Model Context Protocol (MCP)
  • Autonomous agent systems

Module 9: LLM Security

  • GenAI Bill of Materials (GenAIBOM)
  • Security vulnerabilities
  • Attack vectors and defenses

Module 10: Small Models

  • Efficient model architectures
  • Edge deployment strategies
  • Performance vs. accuracy trade-offs

Module 11: Multi-modal Models

  • Vision-language models
  • Audio processing integration
  • Cross-modal applications

Optimization Modules (12-15)

Module 12: MaaS (Model as a Service)

  • API design and deployment
  • Scaling considerations
  • Service level agreements

Module 13: Quantization & Compression

  • Model quantization techniques
  • Compression algorithms
  • Performance optimization

Module 14: Fine-tuning

  • Transfer learning concepts
  • Domain adaptation strategies
  • Best practices discussion

Module 15: Semantic Router

  • Intelligent routing systems
  • Context-aware model selection
  • Advanced implementation patterns
